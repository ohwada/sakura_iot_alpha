### Wiring connection to Arduino

|Sakura IoT|Arduino|
|---|---|
|1 UART TX (O)|D8 or D0|
|2 UART RX (I)|D9 or D1|
|3 GND|GND|
|4 SPI MOSI (I)|D11|
|5 SPI MISO (O)|D12|
|6 SPI SCK (I)|D13|
|7 SPI CS (I)|D10|
|8 GND|GND|
|9 I2C SDA (I/O)|SDA (A4)|
|10 I2C SCL (I/O)|SCK (A5)|
|11 GND|GND|
|12 +5V|+5V|
